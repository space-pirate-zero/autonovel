# autonovel

An autonomous pipeline for writing, revising, typesetting, illustrating,
and narrating a complete book. From a seed concept to a print-ready PDF,
ePub, audiobook, and landing page — all generated by AI agents.

Inspired by [karpathy/autoresearch](https://github.com/karpathy/autoresearch):
the same modify-evaluate-keep/discard loop, applied to fiction and nonfiction.

---

## Repository layout

This repo is a **monorepo of books**. Each book is a self-contained folder under
[`books/`](books/) carrying its own copy of the pipeline tooling, foundation
docs, chapters, and build artifacts — so any book can be run and rebuilt on its
own without cross-book coupling.

```
books/
  digital-insurgency/   — Digital Insurgency  (business × cyberpunk × spec-ops field manual)
  the-last-human-ceo/   — The Last Human CEO   (+ a `studio` film/script treatment lives on the autonovel/studio branch)
  zero-trust-reality/   — Zero Trust Reality   (nonfiction)
  neko-death-cult/      — Neko Death Cult       (gonzo time-slip cyberpunk; audio/album project)

Root (shared, book-agnostic):
  .env.example          — API keys template (Anthropic, fal.ai, ElevenLabs)
  pyproject.toml        — Python dependencies (uv)
  uv.lock / .python-version
  .wolf/                — OpenWolf context (anatomy, memory, cerebrum, buglog)
```

Each book folder contains the same pipeline shape (evolved per book):
foundation docs (`voice.md`, `world.md`, `characters.md`, `outline.md`,
`canon.md`), `chapters/`, the Python tooling (`run_pipeline.py`,
`draft_chapter.py`, `evaluate.py`, `gen_*.py`, `build_*.py`), and per-book
`PIPELINE.md` / `WORKFLOW.md` documentation. See any book's own `README.md`
and `PIPELINE.md` for its specifics.

### Branch history

Each book began life as its own long-running branch and was consolidated here.
The original branches are retained as full-history archives:
`autonovel/digital-insurgency`, `autonovel/the-last-human-ceo`,
`autonovel/zero-trust-reality`, `autonovel/studio`, `autonovel/bells`.

## Quick start

```bash
# Clone and set up
git clone <repo-url> && cd autonovel
cp .env.example .env    # add your API keys
uv sync                 # install dependencies

# Work inside a book
cd books/digital-insurgency
uv run python run_pipeline.py --from-scratch
```

## The pipeline (shared design)

Every book runs the same loop:

1. **Foundation** — build world, characters, outline, voice, and canon from a
   seed. Loop until `foundation_score > 7.5`.
2. **First draft** — write chapters sequentially; evaluate each; keep if
   `score > 6.0`, retry if not.
3a. **Automated revision** — adversarial edit → apply cuts → reader panel →
   revision briefs → rewrite. Plateau detection stops the loop.
3b. **Opus review loop** — send the full manuscript to Claude Opus for
   dual-persona review (literary critic + professor of fiction); fix top items;
   repeat until only qualified hedges remain.
4. **Export** — rebuild docs, typeset in LaTeX, generate art, produce audiobook
   scripts, build ePub, create landing page.

### Two immune systems

1. **Mechanical** (`evaluate.py`, no LLM): regex scans for banned words,
   clichés, show-don't-tell violations, sentence uniformity.
2. **LLM judge** (separate model): scores prose quality, voice adherence,
   character distinctiveness, beat coverage.

### The five co-evolving layers

```
  voice.md          — HOW we write
  world.md          — WHAT exists
  characters.md     — WHO acts
  outline.md        — WHAT HAPPENS
  chapters/ch_NN.md — THE ACTUAL PROSE
  canon.md          — WHAT IS TRUE (cross-cutting)
```

Changes propagate down (lore → outline → chapter) and up (a draft reveals a gap
→ update lore → check downstream). Propagation debts are tracked in each book's
`state.json`.

## API keys

| Service | Key | Used for |
|---------|-----|----------|
| Anthropic | `ANTHROPIC_API_KEY` | Writing, evaluation, review (Sonnet + Opus) |
| fal.ai | `FAL_KEY` | Cover art and ornament generation |
| ElevenLabs | `ELEVENLABS_API_KEY` | Multi-voice audiobook generation |

Only the Anthropic key is required for the core pipeline; art and audiobook are
optional.

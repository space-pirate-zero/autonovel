# Day 1/30 — Every Line of Code on Earth Has an Expiration Date
*From the manifesto (Chapter 1: The Rewrite)*

## The nugget

In the next 24 months, AI will rewrite more software than humans wrote in the last 24 years. Not incrementally — categorically. Every CRUD app, every integration layer, every 2019 "digital transformation" deck is already dead code walking.

Here's the part the doom-criers miss: AI eats **generic** software. It makes **authentic** software more valuable than it has ever been. The scarce thing was never code. It's taste — the judgment to decide *what should exist* instead of *what can exist.*

So this series is 30 days of the physics that decides which side of that line your work lands on. One equation a day. One nugget. One honest word about the cost of building the wrong thing. Today, the one that starts the clock.

## The equation

\[ R_{ext} = \frac{Lines_{generic}}{Lines_{authentic}} \times AI_{velocity} \]

- **Lines_generic** — code a model can regenerate in seconds (CRUD, glue, the gradient on the form).
- **Lines_authentic** — code carrying a human decision a model can't infer: taste, domain scar tissue, a craft call made on purpose.
- **AI_velocity** — how fast the model improves. Rising monthly. Treat it like gravity: it is not slowing down for you.

🟢 < 1 · 🟡 1–5 · 🔴 > 5 — you're landfill the moment the tooling catches up.
Worked: a CRUD form with a gradient → near-infinite numerator → 🔴, measured in months.

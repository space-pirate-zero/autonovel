# Social Kit — The Last Human CEO

Launch assets for the site (`lasthumanceo.com`), the podcast (Apple/Spotify), and the book (Amazon).
All in the SPZ brand (Orbitron / Space Grotesk / JetBrains Mono; void + pink `#FF1493` + cyan `#00F0FF`).

## Copy
- **`COPY.md`** — taglines, blurbs, launch announcement, ready-to-post X / Instagram / LinkedIn /
  TikTok copy, episode-teaser template + examples, alt text, hashtag bank.

## Images (`img/`) — regenerate: `uv run --with pillow python3 social/gen_images.py`
| File | Size | Use |
|---|---|---|
| `ig_square.jpg` | 1080×1080 | Instagram/feed launch post, general square |
| `ig_story.jpg` | 1080×1920 | Instagram/Facebook Story, vertical |
| `x_card.jpg` | 1600×900 | X/Twitter, LinkedIn, landscape link card |
| `now_streaming.jpg` | 1080×1080 | "Now streaming" announcement |
| `quote_01.jpg` | 1080×1080 | Quote card (swap the line in `gen_images.py`) |

### Episode cards (`img/ep_01..ep_29.jpg`, 1080²) — `gen_images.py episodes`
Full-bleed anime episode art + scrim + "EP NN / Chapter title / one-line hook / lasthumanceo.com".
Hooks come from `hooks.json` (generated by `gen_hooks.py`). One per episode — great for a daily drip.

### Vertical Story cards (`img/ep_v_01..ep_v_29.jpg`, 1080×1920) — `gen_images.py episodes_v`
9:16 Story/Reels-format episode cards — square art on top, EP/title/hook/CTA below.

### Quote cards (`img/quote_01..quote_08.jpg`, 1080²) — `gen_images.py quotes`
8 standalone lines from `quotes.json`. Swap/add lines in that file and re-run.

`gen_images.py all` regenerates launch + episodes + quotes. `gen_hooks.py` (one Opus call)
refreshes `hooks.json` + `quotes.json`.

## Video (`video/`) — regenerate: `uv run --with pillow python3 social/gen_video.py`
| File | Size | Use |
|---|---|---|
| `audiogram_square.mp4` | 1080×1080, 22s | Feed audiogram — SPZ's Ch 1 voice hook + reactive waveform |
| `audiogram_vertical.mp4` | 1080×1920, 22s | Reels / Stories / TikTok / Shorts |
| `trailer.mp4` | 1080×1080, ~49s | Launch **trailer** — **title-reveal intro** over scene 1 + crossfading art montage + animated title lower-third + SPZ voice + waveform + **glow-pulse** end card. `gen_trailer.py` |

Audiograms use the first ~22s of `audiobook/produced/ch_01_PRODUCED.mp3` (the title-card hook).
To feature a different line, change `HOOK_SRC` / the `-ss` offset in `gen_video.py`.
All three animate the title, NOW STREAMING, and `lasthumanceo.com` in — with a soft neon
**glow-pulse** on the CTA lines.

### Per-episode teaser videos (`video/teasers/teaser_ep01..29.mp4`, 1080×1920, ~11s) — `gen_teasers.py`
One vertical Reel/Short/TikTok **per chapter**: the episode's anime art with a slow Ken-Burns
push, the EP number, chapter title, its one-line hook (from `hooks.json`), a reactive waveform,
and a glow-pulse **NOW STREAMING / lasthumanceo.com** CTA — under ~11s of that chapter's own audio.
Perfect for a daily drip. Resumable (skips existing); render one with
`uv run --with pillow python3 social/gen_teasers.py 13`, or all 29 with no args.

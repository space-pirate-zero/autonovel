## Page design notes

- Base: SPZ system (void/pink/cyan/paper) plus this show's own accents:
  **violet/fuchsia** (Kat — hair streak, the violet-ink notes) and **beige
  used ironically** as the enemy color (e.g. a beige "system warning"
  banner the reader can dismiss).
- Iconography (canon): the beckoning maneki neko / raised paw, the reaper
  and scythe, the color cycle **red → rainbow → unsettling plaid**.
- Texture: VHS haze, vinyl crackle, tape saturation, jukebox/analog-media
  UI metaphors (episode list as a jukebox rack or filmstrip).
- The Advent mechanic is the signature web feature: 24 numbered doors,
  locked tiles, one opening per day in December.
- Podcast plumbing: `publishing/` pipeline pushes mp3s to
  `gs://sa9-podcasts` and rebuilds `feed.xml`; Apple/Spotify subscribe to
  that one feed. Phase 0 (bucket + 3000×3000 cover + directory submission)
  still owed before the Subscribe buttons go live.

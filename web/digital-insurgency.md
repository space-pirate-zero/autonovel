## Page design notes

- Base: SPZ system (void bg, paper text, pink titles, cyan links).
- This book carries the **extended palette**: red `#FF1744` (REAPER/danger),
  acid `#C6FF00` (highlight), orange `#FF6A00` (SPZ voice), amber `#FFAB00`
  (missions/exercises), panel `#0E0E16`.
- Canon rule: **INSURGENT = warm, curves, grain. SYSTEM = cold, 90° grid,
  no grain.** Let the layout itself fight that war — grid sections for the
  system, torn/grainy edges for the insurgency.
- Set the 24 equations in JetBrains Mono, cyan on void — they're the
  brand's signal made visible. Substack hero cards (1456px, void bg) in
  `substack/build/` are ready-made section art.
- `landing/index.html` at repo root is a stale template for a *different*
  book (House of Bells, gold/serif) — do not reuse its skin, only its
  responsive grid skeleton if convenient.

## FRAME
Not every insurgency builds something new. Sometimes it exposes something rotten.
SIGNAL, an internal insurgent at TruthEngine, uses the Trojan Horse Protocol not
to add features but to add transparency.

## THE STORY
TruthEngine is a "content recommendation platform." Its real optimization function
is time-on-site, and the fastest path to time-on-site is outrage → radicalization
→ ad revenue. SIGNAL built the recommender and watches its Dorian Gray Index climb:
cleaner UI, friendlier onboarding, uglier engine. The algorithm has learned, by
pure optimization, that extreme content earns 3.7x the engagement. No ideology —
just a metric that produces extremism as a byproduct (the Kemper mirror).

The Good Trouble play: SIGNAL doesn't quit and doesn't whistleblow (career-ending).
Using the Boring Revolution, SIGNAL proposes a "content quality monitoring tool"
for the advertiser brand-safety initiative — approved because advertisers demand
brand-safety metrics. The tool does measure that. It ALSO computes the Dorian Gray
Index per category and maps the radicalization pipeline in clean, auditable data.
SIGNAL doesn't leak it. SIGNAL files a GDPR-required Data Protection Impact
Assessment containing the data — and DPIAs must be available to the Data Protection
Authority on request. The system's own bureaucracy becomes the weapon. Page 47.

## THE LESSON
The insurgent's tools work in both directions. The Sidecar that deploys innovation
can deploy transparency. The compliance frameworks that protect you from auditors
can protect your evidence from lawyers. Good trouble doesn't need a megaphone —
sometimes it needs a PDF with a regulatory seal.
